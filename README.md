# Coordination Tax

Interactive visualization of coordination tax and the builder archetype model.

## Overview

This project provides an interactive diagram exploring the concept of "coordination tax" in organizations and how the builder archetype model can reduce overhead in software development teams.

## Contents

- `index.html` -- Main interactive visualization

## Usage

Open `index.html` in a web browser to explore the coordination tax model interactively.
